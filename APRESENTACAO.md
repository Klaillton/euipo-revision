# Roteiro de Apresentacao Tecnica (5-7 min)

## 1. Contexto Profissional

- Resumo de experiencia em desenvolvimento Java.
- Tipos de projeto: sistemas web, APIs, manutencao e evolucao de produto.

## 2. Stack Tecnica

- Linguagem: Java (LTS recentes).
- Back-end: Spring Boot e/ou Java enterprise.
- Persistencia: JPA/Hibernate e SQL.
- DevOps: Docker, CI/CD e monitoramento.

## 3. Projetos e Resultados

- Cite 2 a 3 casos com problema, abordagem e resultado.
- Destaque ganhos de performance, confiabilidade ou produtividade.
- Mostre como foram tomadas decisoes tecnicas e trade-offs.

## 4. Boas Praticas

- Testes automatizados e revisao de codigo.
- Clean Code e principios de design.
- Observabilidade e suporte a producao.

## 5. Fechamento

- Interesse em desafios tecnicos de maior escala.
- Alinhamento com cultura de colaboracao e melhoria continua.

## 6. Perguntas para a Entrevista

- Como e o ciclo de entrega (branching, CI/CD, deploy)?
- Quais sao os principais desafios tecnicos atuais do time?
- Quais tecnologias e praticas sao prioridade nos proximos meses?
